Projet principal

Développement d'un jeu de Labyrinthe en Java

Besoin à satisfaire

Le but du projet était de développer un jeu de plateau fonctionnant uniquement sur terminal. Le joueur devait pouvoir interagir avec un plateau, effectuer des actions et respecter les règles du jeu définies.

Modalités de réalisation

Le projet a été réalisé en binôme. Nous avons utilisé Java pour développer la logique du jeu, des tableaux multidimensionnels pour représenter le plateau et JUnit pour tester certaines méthodes.

Résultat obtenu

Le résultat est un jeu jouable depuis le terminal, avec affichage du plateau, gestion des déplacements, règles de jeu et tests unitaires permettant de vérifier plusieurs comportements.

Recul critique

Si c'était à refaire, j'améliorerais la structure du code en utilisant davantage la programmation orientée objet, j'ajouterais plus de tests automatisés et je développerais une interface graphique avec JavaFX pour améliorer l'expérience utilisateur.

Compétences mises en avant

  • Développer une application simple
  • Utiliser Java et la POO
  • Manipuler des tableaux multidimensionnels
  • Implémenter des règles de jeu
  • Tester avec JUnit
  • Présenter et justifier des choix techniques
